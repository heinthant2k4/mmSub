Features

Everything you need.
Nothing you don't.

Built with one goal: get Myanmar subtitles on your screen with as little friction as possible.

Dual-source search

Searches OpenSubtitles and SubDL simultaneously. Results are merged and deduplicated so you always see the best options first.

Shadow DOM overlay

Subtitles are injected via Shadow DOM — completely isolated from the streaming site's styles. They can never be blocked or broken.

Local file support

Load any .srt, .ass, or .ssa file directly from your device — no search needed.

Frame-accurate sync

Subtitles update every animation frame. Keyboard shortcuts let you nudge timing forward or back without leaving the video.

Auto title detection

Opens the popup with the movie or show title already filled in — reads the page title, og:title, and streaming site DOM selectors.

Display controls

Adjust font size (16–40 px) and vertical position (5–50 %). Settings persist across sessions via browser sync storage.

How it works

Three steps.
Zero friction.

01

Search

Click the myanSub icon. The movie title is auto-detected. Hit Search — results from two subtitle databases appear in seconds.

02

Select

Pick a subtitle result from the list. myanSub downloads and parses it in the background — SRT and ASS formats both supported.

03

Watch

Subtitles appear immediately over the video. Use keyboard shortcuts to nudge sync if needed. That's it.
